# Blinkit Sales Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes Blinkit sales data to provide business insights.

## Features
- Total Sales
- Total Orders
- Total Quantity Sold
- Average Order Value
- Sales by Category
- Top 10 Products
- Sales by Payment Method
- Delivery Status Analysis
- Monthly Sales Trend
- Interactive Slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
